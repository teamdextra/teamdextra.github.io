Dextra

Smart Hand Rehabilitation System

About the Product

Dextra is a low-cost hand rehabilitation system designed to assist patients recovering from stroke or hand injuries. The device enables controlled finger and wrist movements using a mobile application and smart actuators, making rehabilitation accessible and portable.

Technical Limitations

  • Limited to predefined motion ranges
  • Requires Bluetooth connection with the mobile app
  • Not intended to replace professional physiotherapy
  • Designed for supervised or assisted use
